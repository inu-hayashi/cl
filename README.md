## ストレスマネジメントサービス 「Myコーピングリスト」

URL:[https://coping.abrd.site/public/](https://coping.abrd.site/public/)

### 「Myコーピングリスト」について

「気分が軽くなる事をリストにして実行する」というストレスマネジメント法、「コーピング」を簡単に管理、体験できるようにしたサービスです。


### 機能

Laravelで作られており、以下のようなCRUD機能を備えています。

- 会員機能
 - 登録
 - ログイン
- リスト機能
 - 作成
 - 更新
 - 削除
